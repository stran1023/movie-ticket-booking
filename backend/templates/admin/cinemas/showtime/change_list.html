📖 How to manage showtimes:
  • Viewing: Use the date arrows or calendar to jump between days. The timeline visualizes showtimes horizontally across standard working hours.
  • Status: Gray = Draft (hidden), Green = Confirmed (live), Red = Cancelled.
  • Editing: Double-click any showtime block to edit details. Standard single-click lets you select multiple blocks for bulk actions (bottom bar).
  • Auto Generation: Click "Generate Showtimes" to automatically let the system schedule drafts.
  • Manual Creation: Click "Add Showtime" to manually create a showtime.
